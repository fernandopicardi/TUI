# Runnio

Orchestrate multiple Claude Code agents across projects. Parallel. Persistent. Powerful.

## What is Runnio?

Runnio is a desktop application for Windows (and macOS) that lets you run multiple
Claude Code agents simultaneously across different Git projects — each in an isolated
worktree, with persistent terminal sessions, real-time status tracking, and full
Git history visualization.

## Key Features

- **Multi-project**: Open multiple repositories simultaneously
- **Multi-agent**: Run multiple Claude Code agents in parallel per project
- **Persistent sessions**: Navigate between agents without losing context
- **Git History**: Visual branch graph with Code and People modes
- **Agency OS & BMAD**: Native plugin detection for structured workflows
- **Broadcast**: Send prompts to multiple agents simultaneously

## Installation

Download the latest release for your platform:
- Windows: `Runnio-Setup-x.x.x.exe` or `Runnio-x.x.x.exe` (portable)
- macOS: `Runnio-x.x.x.dmg` (coming soon)

## Requirements

- Windows 10 1903+ or macOS 12+
- Git for Windows (includes Git Bash)
- Claude Code CLI installed and authenticated
